# Machine Learning Portfolio

A collection of machine learning models and foundational algorithms built to explore data engineering, predictive modeling, and optimization techniques.

---

## 🛠️ Featured Projects

### 1. Vectorized Multiple Linear Regression (From Scratch)
A lightweight implementation of a parametric learning engine built entirely using raw Python and NumPy, bypassing high-level framework abstractions.

* **Key Concepts:** Matrix transposes and dot products ($\hat{y} = Xw + b$), Mean Squared Error (MSE) cost evaluation, and analytical batch gradient descent with simultaneous parameter updates.
* **Convergence:** Successfully minimized cost from **7.638** down to a stabilized **0.0037**, accurately reconstructing target synthetic coefficients.

### 2. California Housing Price Prediction
An end-to-end machine learning pipeline exploring feature engineering and regression architectures on real-world demographic data.

* **Key Concepts:** Data preprocessing pipelines, feature engineering, handling missing data, and evaluating model performance using metrics like RMSE and $R^2$.
